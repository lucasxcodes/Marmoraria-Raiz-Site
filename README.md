
# 🪨 Marmoraria Raiz - Site

Projeto de site institucional desenvolvido para uma marmoraria, com foco em geração de leads e apresentação de serviços.

## 🚀 Tecnologias utilizadas
- HTML5
- CSS3
- Bootstrap

## 📌 Funcionalidades
- Apresentação de materiais (mármores e granitos)
- Galeria de imagens
- Layout responsivo
- Estrutura otimizada para conversão de clientes

## 🌐 Acesse o projeto
https://lucasxcodes.github.io/Marmoraria-Raiz-Site/

## 💡 Objetivo
Praticar desenvolvimento front-end e criação de páginas com foco comercial.
